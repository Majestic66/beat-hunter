## Recommandations

- Format : PNG avec transparence de préférence
- Taille : 256x256 pixels minimum, format carré idéal
- Style : Photos artistiques, portraits, illustrations cohérentes avec le style du jeu
- Qualité : Haute résolution pour un rendu optimal

## Comment ajouter une photo

1. Créer ou obtenir la photo de l'artiste
2. La nommer exactement comme indiqué ci-dessus
3. La placer dans ce dossier (`public/artist-photos/`)
4. Le jeu l'utilisera automatiquement lors du scouting en France
